# PiGNITHM 街机风格控制器
自制中二节奏风格全尺寸手台
地键 参照chunithm-PSoC方案 使用两块 CY8C4245AXI-483 
天键 参照GSK/GCON2.0方案 使用ATMEGA328P进行ADC转换
使用YubiDeck IO
主控使用 微雪 RP2040 Zero开发板
